# Backend - AI Agents Lab

NestJS backend with Clean Architecture.

## Installation

```bash
npm install
```

## Environment Setup

```bash
cp .env.example .env
# Edit .env with your OPENAI_API_KEY
```

## Development

```bash
npm run start:dev
```

## API Documentation

Once running, visit: http://localhost:3001/api/docs

## Testing

```bash
npm test
npm run test:cov
```

## Build

```bash
npm run build
npm run start:prod
```
