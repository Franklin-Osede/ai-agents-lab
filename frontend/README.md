# Frontend - AI Agents Lab

Angular frontend for interactive agent demos.

## Installation

```bash
npm install
```

## Development

```bash
ng serve
```

Visit: http://localhost:4200

## Build

```bash
ng build
```

## Testing

```bash
ng test
```

## Structure

- `components/` - Page components
- `shared/` - Shared components and services
- `agents/` - Agent-specific components
